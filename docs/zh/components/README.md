# 组件介绍

v-formly 内置了一些基础组件，包括常用的输入框、选择框、单选、多选、下拉框、上传等组件，覆盖了大多数业务，如果这些内置组件不足以满足你的业务，你也可以[自定义组件](TODO)。

::: tip
v-formly 中的组件用法尽可能的与AntDv中的保持一致，少数不一致的地方请参考具体组件API。
